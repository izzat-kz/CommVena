# CommVena
CommVena - Resident Communication and Management Portal

"Comm" signals for Communication, Community, Committee - the three pillars of this system
<br>"Vena" stands for Venation, a structured branching pattern of veins in a leaf. Conveys orderly, neat, interconnected pathway, which is how the portal organizes resident data and workflows
<br>CommVena = Community + Structured Connections

<br>Improve communication in Residential Community between residents and committees, rather than using informal tools like Whatsapp solely, where important notices and complaints tends to buried in chats.
